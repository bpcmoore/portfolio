#Fibonacii

<p>Directory contains multiple implementations for solving the nth fibonacii number
<ul>
    <li>fibonacii.cpp </li>
    <ul>
        <li>Basic recursive implementation</li>
        <li>Exponential time complexity</li>
        <li>Instructions</li>
        <ol>
            <li>Compile</li>
            <ul><li>g++ fibonacii.cpp -o fibonacii</li></ul>
            <li>Run</li>
            <ul><li>./fibonacii n</li></ul>
        </ol>
    </ul>
    <li>fibonacii_top.cpp</li>
    <ul>
        <li>Top down dynamic programming implementation</li>
        <li>Linear time complexity</li>
        <li>Instructions</li>
        <ol>
            <li>Compile</li>
            <ul><li>g++ fibonacii_top.cpp -o fibonacii_top</li></ul>
            <li>Run</li>
            <ul><li>./fibonacii_top n</li></ul>
        </ol>
    </ul>
</ul>
</p>
